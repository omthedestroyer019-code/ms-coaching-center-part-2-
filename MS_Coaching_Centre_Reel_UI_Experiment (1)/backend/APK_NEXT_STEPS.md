# APK next steps

Do not make APK before backend testing. Correct order:

1. Test this PWA demo locally.
2. Connect Supabase and Cloudinary.
3. Host the PWA online.
4. Test on Android Chrome.
5. Use Add to Home Screen to test app-like behavior.
6. After everything works, convert to APK using one of these:
   - Capacitor Android wrapper
   - PWA Builder
   - Trusted Web Activity

For a beginner, PWA Builder is usually easier. Capacitor gives more control but needs Android Studio setup.
